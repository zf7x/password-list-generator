# Password List Generator 🔐

A simple Python tool to generate password combinations using names, numbers, and symbols.

---

## 🚀 Features

* Combines names with numbers and symbols
* Multiple formats (lowercase, uppercase, symbols)
* Generates large password lists automatically

---

## 🛠️ Requirements

* Python 3

---

## ▶️ Usage

Run the script:

```bash
python3 generator.py
```

---

## 📂 Output

The script will generate:

```
passwords.list
```

Example:

```
ziyad1
ziyad@1
Ziyad_1
ZIYAD1
ziyad1!
```

## 📌 Notes

* You can edit names, symbols, and numbers inside the script
* Output file is generated automatically
